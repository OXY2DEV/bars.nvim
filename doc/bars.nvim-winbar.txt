üß≠ Configuration                                       *bars.nvim-winbar.config*

The winbar can be configured via `require("bars").setup({ winbar = { ... } })`
or `require("bars.winbar").setup({ ... })`.

‚ñã ÔêÄ Tip
‚ñã You can use `require("bars").setup({ winbar = false })` or
‚ñã `require("bars.winbar").setup(false)` to disable it!

The configuration table has the following structure.

>lua
	--- Configuration for the winbar module.
	---@class winbar.config
	---
	---@field force_attach? string[] List of `winbar`s to ignore when attaching.
	---
	---@field ignore_filetypes string[] Filetypes to ignore when attaching.
	---@field ignore_buftypes string[] Buffer types to ignore when attaching.
	---
	---@field condition? fun(buffer: integer, window: integer): boolean Additional condition for attaching to windows.
	---
	---@field default winbar.style Default style.
	---@field [string] winbar.style Named style.
<

üé® Styles ~

You can use various styles to easily change how the winbar looks.

>lua
	--[[ Style for the `winbar`. ]]
	---@class winbar.style
	---
	---@field condition? fun(buffer: integer, window: integer): boolean Condition for this style.(unused when style is `default`)
	---@field components winbar.component[] Components for this style.
<

üí° API                                                    *bars.nvim-winbar.api*

You can access this module via,

>lua
	local winbar = require("bars.winbar");
<

üß© High level API ~

The `winbar` module has the following API functions.

‚óá `winbar.setup(config)`, Updates configuration of the winbar.

‚óá `winbar.Start()`, Starts the module and attaches & enables custom winbar for
  all valid windows.
‚óá `winbar.Stop()`, Stops the module and detaches & disables custom winbar for
  all valid windows.

‚óá `winbar.Toggle()`, Toggles the custom winbar for all valid windows.
‚óá `winbar.Enable()`, Enables the custom winbar for all disabled windows.
‚óá `winbar.Disable()`, Disables the custom winbar for all enabled windows.

‚óá `winbar.toggle(window)`, Toggles the custom winbar for `window`.
‚óá `winbar.enable(window)`, Enables the custom winbar for `window`(if it's valid
  & disabled).
‚óá `winbar.disable(window)`, Disables the custom winbar for `window`(if it's
  valid & disabled).

üß© Low level API ~

‚óÜ `winbar.config`, Current configuration.
‚óÜ `winbar.state`, Current state.

>lua
	winbar.state = {
		enable = true,
		attached_windows = {}
	};
<

‚óÜ `winbar.attach(window)`, Attaches to window(doesn't check `ignore_filetypes`,
  `ignore_buftypes` & `condition`).
‚óÜ `winbar.detach(window)`, Detaches from window.
‚óÜ `winbar.update_style(window)`, Causes the winbar style to be updated.
    ‚óá `vim.w.bars_winbar_style`, Winbar style of the current window.
