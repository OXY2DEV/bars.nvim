üß≠ Configuration                                      *bars.nvim-tabline.config*

The tabline can be configured via `require("bars").setup({ tabline = { ... } })`
or `require("bars.tabline").setup({ ... })`.

‚ñã ÔêÄ Tip
‚ñã You can use `require("bars").setup({ tabline = false })` or
‚ñã `require("bars.tabline").setup(false)` to disable it!

The configuration table has the following structure.

>lua
	--- Configuration table for the tabline.
	---@class tabline.config
	---
	---@field condition? fun(): boolean Additional condition for attaching to windows.
	---
	---@field default? tabline.style Default style.
	---@field [string]? tabline.style Named style.
<

üì¶ Components ~

Each style contains one or more components which can be any one of,

>lua
	---@alias tabline.component
	---| tabline.components.bufs Buffer list.
	---| tabline.components.custom Custom text.
	---| tabline.components.empty Empty space.
	---| tabline.components.tabs Tab list.
<

üß© Component: bufs ~

Shows list of opened buffers.

>lua
	--- List of buffers.
	---@class tabline.components.bufs
	---
	---@field kind "bufs"
	---@field condition? fun(): boolean Condition for this component.
	---
	---@field max integer Maximum number of buffers to show.
	---@field truncate_symbol? string Text to use for truncating long buffer name.
	---
	---@field from integer List entry to start showing from.
	---
	---@field separator? string Text used to separate buffers.
	---@field separator_hl? string Highlight group for the separator
	---
	---@field overflow? string Text used to indicate where the buffer list ends(the list is looped).
	---@field overflow_hl? string Highlight group for `overflow`.
	---
	---@field nav_left? string Text for the left navigation button.
	---@field nav_left_hl? string Highlight group for the left navigation button.
	---
	---@field nav_left_locked? string Text for the left navigation button when list scroll is locked.
	---@field nav_left_locked_hl? string Highlight group for `nav_left_locked`.
	---
	---@field nav_right? string Text for the right navigation button.
	---@field nav_right_hl? string Highlight group for the left navigation button.
	---
	---@field nav_right_locked? string Text for the right navigation button when list scroll is locked.
	---@field nav_right_locked_hl? string Highlight group for `nav_right_locked`.

	---@field active bufs.opts

üí° API                                                   *bars.nvim-tabline.api*

You can access this module via,

>lua
	local tabline = require("bars.tabline");
<

üß© High level API ~

The `tabline` module has the following API functions.

‚óá `tabline.setup(config)`, Updates configuration of the tabline.

‚óá `tabline.Start()`, Starts the module and attaches & enables custom tabline for
  all valid windows.
‚óá `tabline.Stop()`, Stops the module and detaches & disables custom tabline for
  all valid windows.

‚óá `tabline.Toggle()`, Toggles the custom tabline for all valid windows.
‚óá `tabline.Enable()`, Enables the custom tabline for all disabled windows.
‚óá `tabline.Disable()`, Disables the custom tabline for all enabled windows.

‚óá `tabline.toggle(window)`, Toggles the custom tabline for `window`.
‚óá `tabline.enable(window)`, Enables the custom tabline for `window`(if it's
  valid & disabled).
‚óá `tabline.disable(window)`, Disables the custom tabline for `window`(if it's
  valid & disabled).

üß© Low level API ~

‚óÜ `tabline.config`, Current configuration.
‚óÜ `tabline.state`, Current state.

>lua
	tabline.state = {
		enable = true,
		attached = false
	};
<

‚óÜ `tabline.attach(window)`, Attaches to window(doesn't check `ignore_filetypes`,
  `ignore_buftypes` & `condition`).
‚óÜ `tabline.detach(window)`, Detaches from window.
‚óÜ `tabline.update_style(window)`, Causes the tabline style to be updated.
    ‚óá `vim.w.bars_tabline_style`, User defined tabline style of the current
      window.
    ‚óá `vim.w._bars_tabline_style`, Calculated tabline style of the current
      window.
‚óÜ `vim.g.bars_tabline_show_buflist`, Set to `true` to switch to buffer list.
